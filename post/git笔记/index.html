git笔记

Mac ssh文件位置：~/.ssh/

配置

  • 修改配置文件

git config –global user.name “名字”

git config –global user.email “邮箱”

  • 查看配置

git config –list

  • git生成ssh

ssh-keygen -t rsa -C “邮箱”

  • gitbash 确认ssh

ssh -T git@github.com

使用

  • 查看状态

git status

  • 手动添加改变

git add

git rm

  • commit

git commit -a -m “描述”

  • push

git push 地址
